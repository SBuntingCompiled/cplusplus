### Circular Buffer

Under this directory there is a simple implementation of 
a circular buffer that utilizes templating as well as 
iterator support. 

It is not meant to be full fledged solution targetting an 
embedded device with guards against ISRs. Instead it is 
rather just a working example to illustrate the concept of
the circular buffer and some features of c++ to help extend 
it. 

To build simply navigate under the circular_buffer directory 
and run:
"make"
